rscss

不失优雅地编写 CSS

构建合理的 CSS 结构。
一系列简单的建议告诉你如何编写可维护的 CSS。

介绍

任何超过 1000 行的 CSS 代码会变得笨拙。并且你最终会掉进这些陷阱：

  • “这个 class 是什么意思？”
  • “这个 class 还在使用吗？”
  • “如果我加了一个新的 class green，会有冲突吗？”

rscss 试图解决上面的问题，它并不是一个框架，rscss 是一些简明地建议，告诉你怎么在任何现代站点或web应用中编写可维护的 CSS 代码。
